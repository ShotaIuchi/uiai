## 変更詳細

### シナリオ例の拡充

**変更前:**

```yaml
name: "ログインテスト"
app:
  android: "com.example.app"
  ios: "com.example.App"
  web: "https://example.com"

steps:
  - id: "起動"
    actions:
      - do: "アプリを起動"
      - then: "ログイン画面が表示されていること"
```

**変更後:**

```yaml
name: "ログインテスト"
app:
  android: "com.example.app"
  ios: "com.example.App"
  web: "https://example.com"

variables:
  email: "test@example.com"
  password: "password123"

steps:
  - id: "起動"
    actions:
      - do: "アプリを起動"
      - then: "ログイン画面が表示されていること"

  - id: "ログイン"
    actions:
      - do: "メールアドレス欄に「(email)」を入力"
      - do: "パスワード欄に「(password)」を入力"
      - do: "「ログイン」ボタンをタップ"
      - then: "ホーム画面が表示されていること"
        strict: true
```
